## How to Add a New ADR

1. Copy the template: `docs/decisions/NNNN-short-slug.md`
2. Use the next sequential number
3. Set status to **Proposed**
4. Fill in Context, Decision, Rationale, Consequences
5. Add an entry to this INDEX
6. After discussion, update status to **Accepted**, **Superseded**, or **Rejected**
